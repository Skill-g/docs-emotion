```bash copy

```

 ``` bash copy
 npm install
 ```

```bash copy
npm run dev
```


# Backend

##### В директории ``` /Server ```

 ``` bash copy
 npm install
 ```

#### После поменяйте в ``` .env ```

##### ``` DATABASE_URL ``` Меняйте данные в зависимости от того какая ваша DB, на проекте используется [prisma](https://www.prisma.io/docs/getting-started/setup-prisma/start-from-scratch/relational-databases/connect-your-database-typescript-mysql)

```bash copy
DATABASE_URL="mysql://johndoe:randompassword@localhost:3306/mydb"
JWT_SECRET="random_key"
```

#### Запушите таблицы или запулите их с помощью [prisma](https://www.prisma.io/docs/getting-started/setup-prisma/start-from-scratch/relational-databases/connect-your-database-typescript-mysql)
```bash copy
npx prisma db push
```

```bash copy
npx prisma db pull
```

#### Backend готов для запуска напишите в ``` cmd ```

```bash copy
node index.js
```

#### Для запуска и фронта и серверной части зайдите в директорию ``` /EMOPROJECT ```
```bash copy
npm run all
```
